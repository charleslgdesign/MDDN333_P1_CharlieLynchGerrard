This project is an investigation into my Screen Time Usage between different applications. I was inspired to research this from the data provided by my Mac Laptop.

This exploration involved 2 variables, Screen Time (Hours) and Days. I manually converted Apple's data into '.CSV' format and imported it into this application using Javascript. Furthermore, JS was used to create the data visualizations via a library called 'P5.js'.
